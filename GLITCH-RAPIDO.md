## 🚀 PASSO 2: Importar Projeto (2 minutos)

### 3.1 Adicionar Variáveis de Ambiente

1. No editor que abriu, procure o arquivo **`.env`** (barra lateral esquerda)
2. Se não existir, click **"New File"** → nome: `.env`
3. Cole isto dentro:

```env
JWT_SECRET=b92eaad9c79e8e5bc3b3626846ec732a1d086ef499db6244c4c300a7509eafea
NODE_ENV=production
```

4. Salva automaticamente ✅

### 3.2 Configurar Banco

1. Click em **"Terminal"** (rodapé da página, ícone de console)
2. Digite:
   ```bash
   npm run setup
   ```
3. Aguarde: ✅ "Banco configurado com sucesso!"
